Plan
1. Säkerställ att onboarding kan navigera till kalendern och tillbaka till hubben.
2. Kräv inbjudningskod för konto och använd den som gate till onboarding.
3. Bygg wizardflöde för upplägg, gård eller stall, resurser, hästar och medlemmar.
4. Gör onboarding hub till enda plats för status och nästa steg.
5. Uppdatera onboarding gate så admin måste bli klar innan appen används fullt ut.
6. Lägg till manuella tester för admin och medlem.

Manuell testlista
1. Gå till onboarding hub och välj upplägg.
2. Skapa gård eller stall och gå tillbaka till hubben.
3. Fyll i resurser och se att status uppdateras.
4. Lägg till häst och se att status uppdateras.
5. Klicka Skapa första pass och landa i kalendern med knapp tillbaka.
6. Skapa pass och gå tillbaka till hubben och se att status uppdateras.
7. Skapa konto utan kod ska blockeras och visa fel.
8. Skapa konto med kod ska koppla till stall.
9. Admin måste slutföra onboarding innan tabs öppnas.
